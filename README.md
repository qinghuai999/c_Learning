# c_learning

## Introduction
本项目为 C 语言学习/实际业务模块的代码工程，目录结构规范，易于扩展和维护。

## 目录结构
- src/           源代码
- include/       头文件
- test/          单元测试
- build/         构建输出
- docs/          项目文档
- scripts/       自动化脚本
- lib/           第三方库
- examples/      示例代码

## 编译与运行
```bash
mkdir build
cd build
cmake ..
make
./c_learning
